# graphql-medellinjs
configuration graphql with express and nodejs

## Prerequisites

- [Git](https://git-scm.com/)
- [nodejs 8.9.4 LTS](https://nodejs.org/es/download/)  

## Let's get started

- git clone https://github.com/IvanVilla1585/graphql-medellinjs.git
- cd graphql-medellinjs
- npm install
- npm start
